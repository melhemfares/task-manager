
# Task Manager

Task Manager app built by following "Node.js / Express Course - Build 4 Projects" video by freeCodeCamp.


## Run Locally

Clone the project

```bash
  git clone https://github.com/melhemfares/task-manager.git
```

Go to the project directory

```bash
  cd task-manager
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
